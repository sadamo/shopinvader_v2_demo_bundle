Product Multi Links (Template)

Este módulo estende a funcionalidade do módulo de vendas para suportar links entre produtos templates.

Este módulo adiciona dois tipos de links:

  • Cross Selling: sugere ao cliente a compra de um produto adicional
  • Up Selling: sugere ao cliente a compra de um produto de categoria superior, um upgrade, etc.

Esses tipos de links são comuns em lojas de e-commerce.

Pode ser usado como base para implementar sincronizações com e-commerce.

Nota Importante

Este módulo faz links entre produtos (templates), não variantes de produtos. Para isso, utilize o módulo Product Multi Link (Variants), no mesmo repositório OCA / e-commerce.
